# code_demo

一个前端练习项目仓库，包含多个独立的 HTML 页面示例，适合快速预览和二次开发。

## 项目结构

- `todo-app/`
  - 经典 Todo List 页面（含输入、列表、状态统计等交互）。
- `software-mall-admin.html`
  - 软件商城管理页面。
  - 包含常用软件列表、搜索筛选、状态筛选、审核通过/上架/下架操作。
- `snake-cyber.html`
  - 科幻风贪吃蛇小游戏。
  - 支持键盘控制、移动端方向按钮、速度调节、最高分记录。
- `bond-trading-system.html`
  - 债券交易系统风格演示页面（用于界面展示与业务原型练习）。

## 如何运行

这是纯静态页面项目，无需安装依赖。

1. 克隆仓库：

```bash
git clone https://github.com/jjfly222/code_demo.git
```

2. 进入目录：

```bash
cd code_demo
```

3. 直接用浏览器打开对应文件，例如：
- `snake-cyber.html`
- `software-mall-admin.html`
- `todo-app/index.html`
