# 项目说明

这是一个前端练习项目，包含多个独立的 HTML 页面示例。

## 项目结构

```
f:\claude-code-demo\
├── todo-app/           # 待办清单应用（渐变玻璃风格）
│   ├── index.html      # 主页面
│   ├── styles.css      # 样式文件
│   └── app.js          # 交互逻辑
├── snake-cyber.html    # 科幻风贪吃蛇游戏
├── software-mall-admin.html  # 软件商城管理后台
└── bond-trading-system.html  # 债券交易系统界面
```

## 文件说明

| 文件 | 描述 |
|------|------|
| `todo-app/` | 经典待办清单应用，支持添加/删除/标记完成、任务统计、本地存储 |
| `snake-cyber.html` | 赛博朋克风格贪吃蛇，支持键盘和移动端控制、速度调节、最高分记录 |
| `software-mall-admin.html` | 软件商城管理页面，含搜索筛选、审核/上架/下架功能 |
| `bond-trading-system.html` | 债券交易系统界面展示 |

## 开发指南

### 运行项目

无需安装依赖，直接用浏览器打开 HTML 文件即可。

### 常用操作

- **预览页面**: 在浏览器中打开任意 `.html` 文件
- **编辑代码**: 修改对应文件后刷新浏览器查看效果
- **查看控制台**: 按 F12 打开开发者工具调试

### 编码规范

- HTML 使用语义化标签
- CSS 使用 BEM 或实用类命名
- JavaScript 使用 ES6+ 语法，保持代码简洁
